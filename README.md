# SmartBathroom

O **SmartBathroom** é um projeto acadêmico desenvolvido com o objetivo de aplicar conceitos de automação, IoT e desenvolvimento de software para o monitoramento inteligente de banheiros.

O projeto foi desenvolvido por **Júlia Wonsick, Carlos Eduardo, João Vinicius e Renan Terre**.

## Sobre o Projeto

A solução permite identificar a falta de insumos nos banheiros, como papel higiênico e papel de mão, utilizando botões físicos, LEDs, buzzer e integração com uma aplicação web.

Quando um botão é acionado, o sistema emite alertas visuais e sonoros para informar a equipe responsável sobre a necessidade de reposição.

## Tecnologias Utilizadas

* Arduino
* Raspberry Pi 4
* Flask
* HTML
* CSS
* JavaScript
* SQLite
* LEDs
* Buzzer
* Botões físicos

## Funcionalidades

* Monitoramento de falta de papel higiênico;
* Monitoramento de falta de papel de mão;
* Alertas visuais com LEDs;
* Alertas sonoros com buzzer;
* Aplicação web para visualização dos status;
* Login com diferentes perfis de acesso;
* Integração entre hardware e software.

## Objetivo

O objetivo do SmartBathroom é tornar o processo de monitoramento e reposição de insumos mais eficiente, contribuindo para um ambiente acadêmico mais organizado, tecnológico e funcional.
